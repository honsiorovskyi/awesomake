# awesomake
Extend your `make` with a set of useful addons.

This repository will contain useful addons, snippets, tricks, shortcuts etc
simplifying life when writing complex Makefile-based build and/or deploy scenarios.

Most of the addons will be presented in a form of `include`able `.mk` files,
providing extra capabilities (e.g. special targets, functions, variables etc 
for the Makefiles.
